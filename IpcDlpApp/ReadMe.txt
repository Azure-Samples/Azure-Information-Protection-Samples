IPC DLP Sample  
====================  
This sample demonstrates how to use AD RMS File API to protect and unprotect files in a DLP solution.  
  
Prerequisites  
=============  
This sample requires Windows 7+.  
This sample requires Visual Studio 2012.
This sample requires MSIPC 2.1 SDK.
  
Sample Language Implementations  
===============================  
C++  
  
To build the sample using Visual Studio 2012:  
================================================  
     1. Open File Explorer and navigate to the  directory.  
     2. Double-click the icon for the .sln (solution) file to open the file in   
     Visual Studio.  
     3. In the Build menu, select Build Solution. The application will be   
     built in the default \Debug or \Release directory.  
  
  
To Run the sample:  
==================  
     1. Open a command prompt.
     2. Navigate to the directory containing IpcDlp.exe.   
     3. Type IpcDlp.exe at the command line.
